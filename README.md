# is-uneven

> Return true if the given number is uneven.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save is-uneven
```

## Usage

```js
var isUneven = require('is-uneven');

isUneven(0);
//=> false
isUneven('1');
//=> true
isUneven(2);
//=> false
isUneven('3');
//=> true
```
